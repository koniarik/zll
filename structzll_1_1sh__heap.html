Detailed Description

template<typename T, typename Acc, typename Compare>
struct zll::sh_heap< T, Acc, Compare >

Skew heap implementation.

Provides API for linking and merging nodes, merging and popping the heap, checking if the heap is empty and accessing the top node of the heap. The top node is the node with the smallest value in the heap according to the comparison function Compare.

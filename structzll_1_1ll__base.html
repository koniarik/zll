Detailed Description

template<typename Derived>
struct zll::ll_base< Derived >

CRTP base class for linked list nodes containing ll_header.

Provides access type to the header of the node and implements move and copy semantics for the node. Provides basic API for the node.

Note that for copy construction to work it has to use non-const reference to the node. This is so we can re-link the copied node into the list.

Detailed Description

template<typename T, typename Acc>
struct zll::ll_list< T, Acc >

Non-owning linked list container, expects nodes to contain ll_header as member.

The nodes are linked together in a doubly linked list, with the first and last nodes accessible through the front() and back() methods.

Type T is the type of the node that contains the header. Type Acc specifies how to access the node's header.

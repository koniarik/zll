Detailed Description

template<typename T, typename Acc>
struct zll::ll_header< T, Acc >

Linked-list header containing pointers to the next and previous elements or the list itself.

Will detach itself from the linked list on destruction.

Type T is the type of the node that contains this header. Type Acc is the access type that provides access to the header of the node.
